## 1. 説明
東洋大学総合情報学部村上真研究室の演習で使用するDockerコンテナ用のDockerfileとビルド・起動スクリプトファイルです．
C, C++, Eigen, OpenGL, OpenCVの環境が構築でき，CLionを使用した3次元コンピュータグラフィックスや画像処理の演習ができます．

## 2. システム要件
- nvidiaのGPUが搭載されている．
- Dockerがインストールされている．
- nvidia-docker2がインストールされている．
